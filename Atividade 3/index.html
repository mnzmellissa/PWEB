.・。.・Ferramentas de Testes de Software.・。.・

O desenvolvimento de software é um processo complexo que exige não apenas a criação de código funcional, mas também a garantia de que ele funcione corretamente em diferentes condições e ambientes. Para isso, podemos utilizar ferramentas desenvolvidas especialmente para testar o sistema.

♡ Testes Funcionais

tem como foco verificar se o software cumpre exatamente o que foi especificado nos requisitos, como funcionalidades visíveis, login e cálculos esperados. Neste teste, o objetivo não é o código mas como ele se comporta ao ser utilizado.

Exemplos de Testes Funcionais:

  • Verificação de login.
  • Teste de um formulário de contato.
  • Validação de um cálculo matemático.

♡ Testes Não Funcionais

Estes testes se concentram em aspectos que não estão diretamente ligados às funcionalidades, como desempenho, segurança e usabilidade. Por exemplo, um teste de carga mede como o sistema se comporta com muitos usuários simultâneos.

Exemplos de Testes Não Funcionais:

  • Teste de Carga
  • Teste de Segurança
  • Teste de Usabilidade
  • Teste de Acessibilidade

Ferramentas de Teste

Existem várias ferramentas para dar suporte a execução de testes.

Ferramentas de Front-end:

  • Selenium
  • Cypress
  • Jest

Ferramentas de Back-end:

  • Postman
  • JUnit
  • PyTest
